# 响应式设计和主题系统演示

## 🎨 主题系统功能

### 三种主题模式
1. **浅色模式** - 经典的白色背景设计
2. **深色模式** - 护眼的深色背景设计  
3. **跟随系统** - 自动跟随操作系统主题设置

### 主题切换
- 点击右上角的主题切换按钮
- 支持循环切换：浅色 → 深色 → 跟随系统 → 浅色
- 主题设置自动保存到本地存储
- 平滑的过渡动画效果

### CSS变量系统
使用CSS自定义属性实现主题切换：
```css
:root {
  --color-bg-primary: #ffffff;
  --color-text-primary: #111827;
  /* ... 更多变量 */
}

.dark {
  --color-bg-primary: #111827;
  --color-text-primary: #f9fafb;
  /* ... 深色主题变量 */
}
```

## 🔧 技术实现亮点

### 1. 主题系统
```typescript
// useTheme Hook
const { theme, resolvedTheme, toggleTheme } = useTheme();

// 自动检测系统主题
const systemTheme = window.matchMedia('(prefers-color-scheme: dark)').matches;

// CSS变量应用
.user-bubble {
  background-color: var(--color-user-bubble);
  color: var(--color-user-text);
}
```

### 2. 响应式Hook
```typescript
// useResponsive Hook
const { isMobile, isTablet, isDesktop, windowSize } = useResponsive();

// 条件渲染
{isMobile ? '简化文本' : '完整文本'}

// 动态样式
className={isMobile ? 'p-3' : 'p-4'}
```

### 3. 自适应组件
```typescript
// 动态尺寸
const buttonSize = isMobile ? 'w-10 h-10' : 'w-12 h-12';
const iconSize = isMobile ? 18 : 20;

// 条件功能
{!isMobile && (
  <div className="desktop-only-feature">
    详细提示信息
  </div>
)}
```
